# Unity Agent Workspace

Public agent-facing configuration for this Unity repo.

## Contents

- `mcp.template.json`: MCP client snippet for Unity's official relay.
- `prompts/`: reusable role prompts for Unity scene/build review.
- `workflows/`: safe operating loops for Unity MCP setup and smoke tests.

Keep this directory public-safe. Do not add local credentials, Unity account data, private screenshots,
or machine-specific project caches.
